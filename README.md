# egg-passport-apple

apple passport plugin for egg

## Install

```bash
$ npm i egg-passport-apple --save
```

## Usage

```js
// config/plugin.js
exports.passportApple = {
    enable: true,
    package: "egg-passport-apple"
}
```

## Configuration

```js
// config/config.default.js
exports.passportApple = {
    // proxy: true,
}
```

see [config/config.default.js](config/config.default.js) for more detail.
